  • Brick-Force Aurora

    January 19, 2025

    Greetings Community,

    We're excited to announce the next version of 2.0.0. The update is available in the Launcher and adds the following features:

    • üõí Fixes for Shop and Inventory
    • Implemented Steam API for matchmaking, networking and friends (Spacewar)
    • Added new, more advanced menu based on Dear ImGui (F6)
    • Added new config options

     

    This marks a big change in how this mod works, as you can now play via Steam with all the features that come with it.

    Steam enables us to have a centralized lobby system where you can create and join lobbies with a lobby list. You can also invite Steam friends or join their lobby directly.

    The new networking also happens through Steam and you won't have to use the old method via raw IP anymore (it's still there if you need it).

    As always, you can grab the update from the launcher, but to enable Steam play, some additional steps are required:

    • Install and run Steam, create or login to your Steam account
    • Install 'Spacewar' (Proxy for Steamworks API access) to your Steam Library by opening steam://install/480/ in your browser while Steam is running
    • Run Brick-Force through BfLauncher.exe or BrickForce.exe while Steam is running
    • Steam should show you as playing 'Spacewar'
