 About Brick-Force Aurora

Welcome to Brick-Force Aurora

Brick-Force Aurora is a full rewrite of the original server, now featuring enhanced stability and a wide array of new functionalities. This project is built with love by fans to keep the magic of Brick-Force alive.

Experience multiplayer over VPN or LAN, host matches directly from within the game, and customize your gameplay with the newly added inventory editor. It's everything you loved about Brick-Force, and more!

Features

Play multiplayer over VPN or LAN.

Host matches seamlessly from within the game.

Load any original map file.

Interactive and destroyable bricks such as glass, crates, cannons, and trains.

Classic team configurations and match setup options.

Various bug fixes over the original game.

How to Play

Getting started with Brick-Force Aurora is simple:

Download the latest version.

Install the game and ensure updates are applied (can be disabled in launcher settings).

Connect with other players via VPN or LAN (e.g., Radmin VPN).

Allow firewall exceptions if prompted.

Set the host's IP in the game settings or use 127.0.0.1 for solo testing.

Host clicks "Host Match"; others click "Join Match."

Start the game, create a room, and get ready to play!
